< 프로젝트 관리 >

* 팀명 : 꽃길만 걷(조)

* 팀장 : 김민영 

* 구성원 : 서지연, 김여밈, 김민영, 최여정, 한아름 

* 주제 : 문화센터 

* 역할분담 : 
	서지연 : 마이페이지
	김여밈 : 로그인(+강좌)
	김민영 : 관리자 / 오시는길
	최여정 : 커뮤니티
	한아름 : 강좌

* 개요 :
	기존 문화센터 홈페이지에서는 강좌 신청시 복잡한 화면구성으로 신청까지 어려움이 있어,
	문화센터를 직접 방문하여 신청하는 회원이 많았습니다.
	이에 누구나 손쉽게 접근할 수 있는 홈페이지를 구축하려 합니다.

* 구현기능 : 

0) 벤치마킹 사이트
	현대백화점
	롯데마트
	서대문 청소년 수련관

1) 문화센터 소개
	1. 대표인사말
	2. 시설안내
	3. 강사소개
	4. 오시는길(다음지도)

2) 프로그램
	1. 강좌안내(리스트)
	
	2. 강좌시간표( 달력 / 찜하기, 장바구니, 신청하기 )
		- 폐강(정원미달) / 접수마감(정원초과)
		- 신청하기(지역주민만 가능하게)

3) 이용안내
	1. 회원이용규정
	2. 셔틀안내

4) 커뮤니티
	1. 공지사항
	2. Q&A
	3. 강좌후기
	4. 포토갤러리

5) 로그인
	SNS 로그인(보류)

6) 회원가입

7) 마이페이지
	1. 나의 정보
	2. 회원정보 수정
	3. 나의 강좌
	4. 결제 및 취소내역
	5. 나의 게시글

8) 관리자
	1. 강사
		- 회원정보 리스트
		- 본인 근태관리(휴가, 출근퇴근)
		- 본인 수업안내
	2. 관리자
		- 강사 급여관리(계좌, 은행, 급여, 상여)
		- 회원 수강신청목록(누적)

9) 총정리
	로그인 : 로그인, 로그아웃, 회원탈퇴, 회원가입, 회원정보수정, ( 아이디, 비밀번호 찾기, SNS 로그인 )
	게시판 : 나의 게시글, Q&A(강사,관리자게시판), 강좌후기(+포토갤러리), 공지사항(관리자게시판)
       	       : 강좌시간표, 강좌목록, 강좌소개글, 센터안내(오시는길), 급여관리, 근태관리

* 설계의 주안점 : 
	1. 사용자 편의성
		- 강좌 리스트를 달력 형태로 제공
	2. 관리의 효율성
		- 업무정보 공유 및 자료 통합 검색 가능

* 사용기술 및 개발환경 : 

	1. JAVA
	2. JSP
	3. ORACLE DB
	4. 인코딩 = UTF-8
	5. 브라우저 = 크롬
	6. IDE = 이클립스
	7. JDK 1.8.91
	8. APACHE TOMCAT v7.0
	9. 프로젝트관리 = github(보류)
